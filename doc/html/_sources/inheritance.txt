.. _inheritance:


Inheritance diagrams
********************

Inheritance diagrams can generate the uml class inheritance diagrams which explains to people easily.

By seeing this diagrams, we can understand how the classes are inherited from one to another in many ways.

.. note:: Inheritance means derive all the properties of one or more parents(super) class to its child(sub) classes.
